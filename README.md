# Shopping List App

간단하고 직관적인 쇼핑 리스트 웹 애플리케이션입니다.

## 기능

- 쇼핑 아이템 추가/삭제
- 구매 완료 체크
- 완료된 항목 일괄 삭제
- LocalStorage를 이용한 데이터 저장 (새로고침해도 유지)
- 반응형 디자인

## 사용 방법

1. `shopping-list.html` 파일을 브라우저에서 열기
2. 입력창에 구매할 아이템 입력 후 "추가" 버튼 클릭 또는 Enter 키
3. 아이템 왼쪽 원을 클릭하여 구매 완료 표시
4. 아이템에 마우스를 올리면 나타나는 × 버튼으로 삭제

## 기술 스택

- HTML5
- CSS3 (Gradient, Animation, Flexbox)
- Vanilla JavaScript
- LocalStorage API
